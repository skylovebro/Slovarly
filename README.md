# Slovarly
Словарли - приложение для пополнения словарного запаса поможет вам легко и эффективно изучать новые слова. Вы сможете добавлять свои собственные слова. Разнообразные тесты позволят закрепить знания и проверить свои навыки. Удобный интерфейс и увлекательный подход сделают обучение языку простым и интересным
